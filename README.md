# WanKotlin
使用Kotlin语言开发的WanAndroid客户端，使用了 AndroidX 库和 RxJava、Retrofit、OkHttp 等开源技术。

## 目前已完成的功能：

* 首页Banner
* 首页推荐文章
* 搜索热词
* 搜索热词本地保存
* 项目分类
* 登录、注册和退出
* 公众号Tab
* 收藏文章（线上收藏和本地收藏）
* 取消收藏
* 搜索功能

**后续会持续更新未完成的功能，如大家有歧义的地方，可提Issues，谢谢！**
